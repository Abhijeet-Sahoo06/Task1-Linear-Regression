# Linear Regression - Task 1

This project was completed as part of my internship at **Maincraft**. The goal of this task was to understand and implement a Linear Regression model using Python to predict salary based on years of experience.

## About the Project

In this project, I:
- Loaded the dataset using Pandas.
- Explored and understood the data.
- Visualized the relationship between years of experience and salary.
- Split the dataset into training and testing sets.
- Trained a Linear Regression model.
- Predicted salary values.
- Evaluated the model and visualized the regression line.

## Tools & Libraries Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Files Included

- `task1_ml_linear_regression.ipynb` – Complete implementation of the project.
- `Salary_Data.csv` – Dataset used for training and testing.

## Conclusion

This task gave me practical experience in building a basic machine learning model. It helped me understand the complete workflow of a Linear Regression project, from loading data to training the model and making predictions.
